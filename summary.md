# 최근 Notion 생성 페이지 요약

현재 설정된 Notion 연동(MCP 서버) 기능의 한계로 인해, 페이지의 내부 본문 텍스트(Block Children)를 직접 불러올 수 없어 **가장 최근에 생성 및 수정된 페이지들의 제목과 메타데이터를 기반으로 요약**을 제공합니다.

## 최근 생성 및 수정된 문서 목록

1. **주간 할 일 목록**
   - **유형:** 페이지
   - **생성 일자:** 2026-03-04
   - **최근 수정 일자:** 2026-05-06
   - **요약:** 주간 단위의 할 일(To-Do)을 관리하기 위해 최근에 업데이트된 페이지입니다. 

2. **월별 가계부**
   - **유형:** 페이지
   - **생성 일자:** 2026-03-04
   - **최근 수정 일자:** 2026-05-06
   - **요약:** 월별 자금 흐름을 기록하는 가계부 문서로 최근 확인되었습니다.

3. **가계부 세부 수입 항목 (데이터베이스)**
   - **월급:** 3,500,000원
   - **생일 용돈:** 150,000원
   - **부업:** 450,000원

> **참고:** Notion API 연동 시 페이지 본문(Block Children)을 읽어오는 기능이 추가되면 내부 텍스트에 대한 더 상세한 요약이 가능합니다. 현재는 페이지 제목과 속성(Property)만 조회할 수 있습니다.
